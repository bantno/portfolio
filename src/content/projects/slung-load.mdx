## Problem & Approach

Slung loads cause problematic oscillations during quadcopter flight — the pendulum dynamics of a suspended payload excite the vehicle's attitude controller, degrading tracking performance and risking instability during aggressive maneuvers.

Implemented input shaping on a slung load quadcopter using PX4-ROS2 offboard control. The input shaper modifies the commanded trajectory to suppress the payload's natural frequency, reducing oscillation amplitude without requiring payload state feedback.

SITL testing in Gazebo reproduced the quadcopter and slung load dynamics, validated with real-world flight testing. The same ROS2 offboard control node ran in both environments.
