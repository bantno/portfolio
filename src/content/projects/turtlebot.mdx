## Problem & Approach

Implemented autonomous maze navigation for a Turtlebot using computer vision to detect and interpret directional signs. The perception pipeline processes camera frames to detect signs, classify their directional content, and feed navigation decisions to the motion planner.

The system was developed entirely in Gazebo simulation, then validated on real Turtlebot hardware — the same perception and navigation pipeline running in both environments with no code changes. This sim-to-real transfer validated the robustness of the vision processing pipeline against real-world lighting variation and sensor noise.
